
# SQL Delivery Analysis
This project analyzes delivery data using SQL queries to identify delays, delivery status, and regional performance.

## Tecnologias
- SQL
- Deepnote
- CSV datasets
  
## Project Objective
Analyze delivery performance and identify problematic deliveries.

## Análises realizadas
- Filtros com WHERE
- Operadores AND / OR
- Uso de IN e NOT IN
- Subqueries
- Análise de entregas problemáticas
